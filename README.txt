Hello and welcome to Hangman 1.0 !

To run the Hangman game in a terminal:

- Open a new terminal window.
- Go to the cd directory the .jar file is located in.
- Type: java -jar mc222ap-1dv600.jar

The game should now be running.


To run the Hangman game in an IDE with Java installed.

- Open the project mc222ap-1dv600 in the IDE
- Open the HangmanMain class
- Run the application

The game should now be running in the IDE terminal.
